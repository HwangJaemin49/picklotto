 # 🎰 PickLotto (픽로또)                                                                                   
  2                                                                                                     
  3 **PickLotto**는 직관적이고 세련된 디자인을 갖춘 프리미엄 로또 번호 생성기 웹 애플리케이션입니다. 단순한  
    랜덤 생성을 넘어 사용자가 원하는 조건을 설정하고 생성된 번호를 관리할 수 있는 기능을 제공합니다.    
  4                                                                                                           
  5 ## ✨ 주요 기능                                                                                           
  6                                                                                                           
  7 - **다양한 생성 모드**:                                                                                   
  8   - **완전 랜덤**: 클릭 한 번으로 무작위 번호 6개 추출.                                                   
  9   - **조건 설정 (반자동)**:                                                                               
 10     - **고정수**: 반드시 포함할 번호 선택 (최대 5개).                                                     
 11     - **제외수**: 결과에서 배제할 번호 선택 (최대 10개).                                                  
 12 - **시각적 애니메이션**: 로또 공이 튀어나오는 듯한 애니메이션 효과와 공식 로또 색상 적용.                 
 13 - **내 번호함 (저장 기능)**: 생성된 번호를 로컬 스토리지에 저장하여 나중에 다시 확인 가능 (날짜별 정렬).  
 14 - **반응형 디자인**: 모바일과 데스크톱 환경 모두에 최적화된 UI/UX.                                        
 15                                                                                                           
 16 ## 🛠 기술 스택                                                                                           
 17                                                                                                           
 18 - **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (Vanilla JS)                                        
 19 - **Storage**: Browser LocalStorage (별도의 DB 없이 기기에 저장)                                          
 20 - **Design**: Premium Navy & Gold Theme                                                                   
 21                                                                                                           
 22 ## 🚀 시작하기 (Deployment)                                                                               
 23                                                                                                           
 24 이 프로젝트는 단일 HTML 파일로 구성되어 있어 배포가 매우 간편합니다.                                      
 25                                                                                                           
 26 ### 1. 로컬에서 실행                                                                                      
 27 1. `index.html` 파일을 다운로드합니다.                                                                    
 28 2. 브라우저(Chrome, Edge 등)로 파일을 엽니다.                                                             
 29                                                                                                           
 30 ### 2. GitHub Pages 배포 방법                                                                             
 31 1. GitHub에 새 저장소(Repository)를 생성합니다.                                                           
 32 2. `index.html`과 `README.md` 파일을 업로드합니다.                                                        
 33 3. 저장소의 **Settings > Pages** 메뉴로 이동합니다.                                                       
 34 4. Build and deployment의 Branch를 `main`으로 설정하고 **Save**를 누릅니다.                               
 35 5. 제공되는 URL을 통해 접속합니다.
